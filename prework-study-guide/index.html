HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The big elements such as head, body, and main are considered parent elements.
  • The small elements nestled into the bigger ones such as sections, headers, and lists are called child elements.
  • HTML acts as the frame that the website is structured on.

CSS

  • CSS is the stylization aspect of building a website.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can determine colors in CSS by semantic elements or hex codes.
  • CSS is also used to determine the sizing and looks of headers, footers, images, and headers.

Git

  • git status: checks what branch you are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: moves you to an existing branch
  • Git works locally in connection with your CLI, and github is then synced and stores your information on the cloud for others to view or edit your work.
