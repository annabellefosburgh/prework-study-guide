HTML

    The head element contains information about the webpage. The body element represents the visible content shown to the user. The big elements such as head, body, and main are considered parent elements. The small elements nestled into the bigger ones such as sections, headers, and lists are called child elements. HTML acts as the frame that the website is structured on.
